Ldap2date.js

Do you have a string in RFC 4517 Generalized-Time syntax, when you really just want a Date? Or perhaps you need some Generalized Time, but have a Date instea. If so, this may help you.

Usage

ldap2date.parse(time)

Turns Generalized-Time strings into Date objects. The value null will be returned if the string cannot be parsed.

  • Time - A LDAP Generalized-Time string will be returned if the time string cannot be parsed into a valid Date object.

Enter a Generalized Time string:

Date: Thu, 28 Feb 2013 19:27:06 GMT

var date = ldap2date.parse('20130228192706Z') date.toUTCString() // 'Thu, 28 Feb 2013 19:27:06 GMT'

ldap2date.toGeneralizedTime(date)

Turns a Date object into Generalized-Time string.

  • date - A date object

var date = new Date('Thu, 28 Feb 2013 19:27:06 GMT') ldap2date.toGeneralizedTime(date) // '20130228192706Z'
